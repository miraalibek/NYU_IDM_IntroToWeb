jQuery uses a format, $(selector).action() to assign an element(s) to an event.

Find elements using selectors

$("p") -- Select all p elements

$("#my_id") -- id selector

$(".my_class") -- class selector

$("div p") -- selects all p element inside a div

more selectors
Do something using jQuery methods

$(".test").hide(); -- Hide .test elements

$(#test).toggleClass("myClass") -- adds and removes class

$("div").first().css("background-color","green") -- turns background green for first div element
