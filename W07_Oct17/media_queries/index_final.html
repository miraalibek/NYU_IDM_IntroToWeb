Responsive Design

To build a respoinsive site means creating a UI responding to different viewports/devices

@media queries

Changing page layout to accomodate different widths of devices requires special code. Media queries is the main way to do that, but you also have to have the appropriate meta tag in your html pages.

min-width

min-width is used when one designs for mobile first and then use media queries to build responsive patterns.

max-width

max-width is used when the initial code is done for desktop, and then sized down with @media code
