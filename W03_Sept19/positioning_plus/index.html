Positioning

There is static (default), relative, absolute and fixed positioning.
Find out more >

Position Relative

An element with position: relative; is positioned relative to its normal position.Use top, right, bottom, and left properties to move div relative to its current position.

Absolute Positioning

The element is positioned relative to its container(ancestor). If this element has no positioned ancestors, it uses the document body.

Fixed

An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled.
